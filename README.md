# py02

エラー処理と例外にフォーカスした演習ステージです。
入力の検証や想定外の状態に対する対応、例外クラスの設計を学びます。

## どんなことをやるか

- `ft_first_exception.py`: 例外処理の基本構文 `try/except`
- `ft_different_errors.py`: 複数種類の例外を分けて処理する方法
- `ft_custom_errors.py`: 独自の例外クラスを定義して使う
- `ft_finally_block.py`: `finally` で後始末を確実に行う
- `ft_raise_errors.py`: 意図的に例外を発生させる設計
- `ft_garden_management.py`: 例外処理を使った状態管理

## 目的

- 安定したプログラムを書くための例外処理技術を習得する
- バリデーションとエラーメッセージの扱い方を理解する
- 失敗する可能性のある処理を安全に実装する

## 実行方法

1. `cd py02/exN`
2. `python3 ft_*.py`

`py02` は、実用的な Python プログラムに必要な堅牢性を養うステージです。
