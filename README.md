# py02

このフォルダでは、エラー処理と例外にフォーカスした演習をします。
入力の検証や想定外の状態に対する対応、例外クラスの設計を通して、より堅牢なプログラムを書けるようになります。

## 何をする課題か

- `ex0/ft_first_exception.py`: `try/except` を使って、入力ミスでプログラムが止まらないようにする
- `ex1/ft_different_errors.py`: 数値変換エラーや範囲外エラーなど、種類ごとに処理を書く
- `ex2/ft_custom_errors.py`: 自分で例外クラスを作って、独自のエラー条件を追加する
- `ex3/ft_finally_block.py`: 例外が起きてもファイルや接続を閉じるような後始末を書く
- `ex4/ft_raise_errors.py`: 条件が満たされないときに、わざとエラーを出す
- `ex5/ft_garden_management.py`: 例外処理を使って、ガーデンの状態を安全に管理する

## ここで学ぶこと

- 例外を捕まえて処理する
- エラーの種類に応じて分けて扱う
- 独自の例外クラスを作る
- 失敗する可能性のある処理を安全に実装する

## 実行方法

1. `cd py02/exN`
2. `python3 ft_*.py`

`py02` は、実用的な Python プログラムに必要な堅牢性を養うステージです。
